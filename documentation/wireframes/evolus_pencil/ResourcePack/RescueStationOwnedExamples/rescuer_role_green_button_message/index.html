Ξ∞×
Alice has pressed the Green Button.

RescueStation is sending a Green Button Message to Bob.

While the message is being sent, Alice can't do any more interaction with the app.

If the app is sending a message when a message comes in from someone else, the UI for the outgoing message is cancelled, but the message is still sent.
The UI for the incoming message is then displayed.


In the 'Sending' phase, the app is trying to contact the server, to upload the message content. It will then wait for the server to send the message on to Bob's phone.
On receiving the message, Bob's phone will send a system message back (an 'ack') to the server.
The server will send a system message (an 'ack') back to Alice's phone.
The App receives the 'ack' message.
This guarantees that the App 'knows' the message has got to Bob.
